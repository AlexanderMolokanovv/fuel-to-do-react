# Проект: Нейросетевое проектирование состава топливных композиций

Проект представляет собой веб-приложение для расчета характеристик топливных композиций с использованием нейросетевых алгоритмов. Состоит из фронтенд-части (React) и бэкенд-части (API для обработки расчетов).

## Ссылки

- **Деплой**: [gmkv.nomoredomains.rocks](https://gmkv.nomoredomains.rocks/)
- **Репозиторий**: [github.com/AlexanderMolokanovv/fuel-to-do-react](https://github.com/AlexanderMolokanovv/fuel-to-do-react)
- **IP-адрес сервера**: `158.160.55.39`

## Описание

Приложение позволяет вводить параметры летательного аппарата (тип, масса, объем бака, полезная нагрузка и др.), отправлять данные на сервер для расчета КПД топлива, дальности полета и стоимости, а затем отображать результаты.

## Директории

- `/` — Главная страница для ввода параметров летательного аппарата.
- `/results` — Страница с результатами расчета.
- **Вспомогательные директории**: Создаются разработчиком по необходимости (например, `/api`, `/components`).
